# 用途

1. 安卓 ？安卓90%开发市场已经在2019年转变为kotlin语言，Java占有逐步降低
2. 桌面开发？桌面开发打多选择微软的相关技术，开发场景不多
3. 后端？国内外现在大部分后端使用的是Java，其占有很大的份额，很多Java程序员也是从事这个工作
